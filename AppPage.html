Communication has always been crucial to mankind. From carving into stones to the latest chat apps, human seeks for better ways to talk to each other. Although many chat applications fulfill the customer's need in this regard, we still see a big gap between ideal conversations through mobile chats and current chat applications.
This story led us to develop a new chat application that helps people communicate in a more organized way allowing people to quickly get the information they need, with less risk of spam. SplitChat, in this regard, can keep conversations neat and organized within a specific group while the subject defines what is going to be discussed in each thread page. This way, the user can create his/her own thread page next to the general chat page and invite the target audience to obtain the requested information.
